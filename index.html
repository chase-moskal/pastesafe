pastesafe

open source web encryption app

securely communicate in public via links

pastesafe allows you to securely communicate sensitive data across any open medium, like youtube comments, or anything else

pastesafe is

  • 100% open source pastesafe on github
  • 100% clientside-only operation (no servers)
  • RSA-OAEP 4096 encryption

all pastesafe communications are invite-based

  1. the person who wants to receive data initiates by sending an invite-link
  2. the link allows the opener to encrypt a message in a link they can send back
