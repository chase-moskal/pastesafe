pastesafe prototype

open source encryption

securely communicate over any medium via encrypted links

all pastesafe communications are initiated by an invite

  1. bob wants secret info from alice. so, bob creates a pastesafe session and sends her the invite link
  2. alice clicks the link, and is prompted to enter her secret info
  3. pastesafe packs the secret info into an encrypted link which only bob can decode
  4. alice sends back this response link, and bob decodes it

pastesafe is

  • public key cryptography
  • RSA-OAEP 4096 encryption
  • 100% open source on github
  • 100% serverless, clientside-only webcrypto
