it was originally built to facilitate two people sharing sensitive information, like passwords, over an untrusted medium like any internet forum

so pastesafe is a medium-agnostic end-to-end encryption app

all pastesafe communications are initiated by an invite

  1. bob wants secret info from alice. so, bob creates a pastesafe session and sends her the invite link
  2. alice clicks invite link, and is prompted to enter her secret info
  3. pastesafe packs the secret info into an encrypted link which only bob can decode
  4. alice sends back the response link, and bob clicks it to decode

more about pastesafe

  • 100% open source on github
  • 100% serverless clientside webcrypto
  • asymmetric public key cryptography, rsa-oaep 4096
