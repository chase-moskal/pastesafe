pastesafe lets you talk secretly in public

exchange private information via encrypted links

all pastesafe communications are initiated by an invite link

  1. bob wants to receive secret info from alice. so, bob creates a pastesafe session and sends her the invite link
  2. alice clicks the invite link, and is prompted to enter her secret info for bob
  3. pastesafe packs the secret response into an encrypted link, which only bob can decode
  4. alice sends back this new response link, and bob clicks to decode it

more about pastesafe

  • 100% open source on github, let's collaborate!
  • 100% serverless clientside webcrypto, keys in localstorage
  • asymmetric public key cryptography, rsa-oaep 4096
